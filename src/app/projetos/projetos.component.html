EcoMaps Lorena-SP

Aplicativo para encontrar os ecopontos da cidade de Lorena-SP. Desenvolvido através do framework Angular, utilizando componentes, diretivas, funções, e bibliotecas para geração do conteúdo. Utilizando linguagem TypeScript e biblioteca do Bootstrap, além de HTML e CSS para construção e organização do aplicativo e sua responsividade.

Tecnologias utilizadas

HTML5 - CSS3 - TYPESCRIPT - ANGULAR - Bootstrap

Take Your Umbrella

Interface desenvolvida em Python com a biblioteca Tkinter, que consome uma API disponibilizada gratuitamente do Clima Tempo (site de previsões meteorológicas). Trás informações a respeito do tempo referente a cidade de Lorena, como probabilidade de chuva e uma breve descrição de como será o dia.

Tecnologias utilizadas

Python - Tkinter

Jogo da Velha

Jogo da velha em linguagem Java e orientado a objetos. O jogo foi desenvolvido através de classes, interfaces, objetos, herança, polimorfismo, encapsulamento e métodos, além de envolver interação com o usuário através do console, com entrada de dados, analise de estado do jogo, analise e validação das jogas e analise do placar. Níveis de dificuldades também foram implatados possuindo lógicas diferentes, e definido a estratégia da máquina à cada jogada.

Tecnologias utilizadas

Java
